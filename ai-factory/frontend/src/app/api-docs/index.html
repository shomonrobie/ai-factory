API Reference

Complete REST API documentation for FactoryOS AI.

Interactive Documentation: http://localhost:4001/docs

Authentication

curl -X POST http://localhost:4001/api/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@factoryos.ai","password":"admin123"}'

Batches API

Create Batch

curl -X POST http://localhost:4001/api/v1/batches/ \
  -H "Content-Type: application/json" \
  -d '{"batch_number": 1, "prompt": "Create a todo app"}'

List Batches

curl -X GET http://localhost:4001/api/v1/batches/

Agents API

List Agents

curl -X GET http://localhost:4001/api/v1/agents/

Generate Code

curl -X POST http://localhost:4001/api/v1/agents/generate \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Create a REST API", "entity_name": "Product"}'

Admin API

Get Users

curl -X GET http://localhost:4001/api/v1/admin/users

Create User

curl -X POST http://localhost:4001/api/v1/admin/users \
  -H "Content-Type: application/json" \
  -d '{"name": "New User", "email": "user@example.com", "tier": "free"}'

Orchestration API

Create Plan

curl -X POST http://localhost:4001/api/v1/orchestration/plan \
  -H "Content-Type: application/json" \
  -d '{"goal": "Build an e-commerce platform", "context": {}}'
